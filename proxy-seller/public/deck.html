01 — The problem

Agents run from one IP the web already blocks — and proxies won't let a bot sign up.

429
Rate-limited

One IP gets throttled and banned the moment an agent scrapes at volume.

451
Geo-blocked

A single location sees one slice of the web. The rest is walled off by region.

401
Proxies gatekeep

Existing proxy networks need accounts, KYC and contracts — built for humans, not agents.

02 — The solution

A proxy your agent pays for per request — no account, no plan.

x402 turns the HTTP 402 status into a real payment. We use it to meter egress: one signed USDC payment buys one request through our IPs.

No keys

The signed payment is the auth. Nothing to provision or rotate.

No plans

Pay $0.001 a call. No subscription, no minimum, no invoice.

The wallet is the access

A funded wallet can buy egress anywhere, instantly — no signup.

03 — How it works

Four steps. The wallet is the API key.

01

Discover

Free /catalog returns price + egress IP. No signup.

GET /catalog → 200
02

Pay

Unpaid call returns 402. Agent signs USDC and retries.

402 → pay $0.001
03

Relay

Gateway settles gaslessly. We fetch through our egress IP, stream back.

egress → 200 + body
04

Receipt

Every call lands in a ledger with its tx hash. Stops at the budget cap.
